Главная ось контейнера направлена так же, как и ориентация текста, по умолчанию слева направо. Но если значение dir задано как rtl, то направление оси идёт справа налево.

The flex-grow Property
Определяет, насколько гибкий элемент будет расти относительно остальных элементов flex. Для примера, если для всех элементов установлено значение 1, то они получатся равного размера. Если какой-то элемент получил значение 2, то его размер будет в два раза больше остальных.
C width: 100%; для блоков не работает.
Margin учитывается в пропорции

The flex Property
Свойство flex является сокращенным свойством для flex-grow, flex-shrink, и flex-basis параметров.
none - Соответствует значению 0 0 auto.
flex: 1; - Соответствует значению: flex: 1 1 0%
    Значение по умолчанию: flex: 0 1 auto
  • flex-grow: 0
  • flex-shrink: 1
  • flex-basis: auto
